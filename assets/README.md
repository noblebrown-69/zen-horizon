# Assets for Zen Horizon Dashboard

This folder contains all the assets needed for the Zen Horizon Dashboard application.

## Required Files

### Fonts
Download and place these fonts here:

- `NotoSerifJP-Regular.ttf` - [Noto Serif JP Regular](https://fonts.google.com/specimen/Noto+Serif+JP)
- `NotoSansJP-Regular.ttf` - [Noto Sans JP Regular](https://fonts.google.com/specimen/Noto+Sans+JP)

### Background Images
Download 8 Japanese art images from Unsplash and name them exactly as follows:

- `bg1.jpg`
- `bg2.jpg`
- `bg3.jpg`
- `bg4.jpg`
- `bg5.jpg`
- `bg6.jpg`
- `bg7.jpg`
- `bg8.jpg`

Use these search terms on Unsplash for appropriate zen-themed backgrounds:
- Japanese landscape
- Zen garden
- Sakura trees
- Mountain temples
- Traditional Japanese art
- Bamboo forest
- Koi pond
- Tea ceremony

Ensure images are high resolution (at least 1920x1080) for best fullscreen display.

## Notes

- All files must be in the root of this `assets/` folder
- The application expects exact filenames as listed above
- Images should be in JPG format
- Fonts should be TTF format from Google Fonts
