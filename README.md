# Zen Horizon Dashboard

Portable old-school C++ idle dashboard for Linux (Ubuntu/Mint XFCE).  
One binary. Dropbox-ready. Runs on any of your machines.

## Features
- Live stocks (customizable via S key)
- Rotating zen quotes (Bruce Lee plateau included)
- Falling sakura particles
- Japanese art backgrounds cycling
- Lower-left digital clock
- Lower-right weather (Fahrenheit + % chance of rain, customizable via W key)
- Fullscreen (F11 toggle), ESC to quit

## One-time setup
```bash
sudo apt install zenity libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev libcurl4-openssl-dev
```
